# 프로젝트 개요: 2022 개정 교육과정 성취수준 관리 시스템

## 1. 개요
2022 개정 교육과정의 교과별 성취기준에 따른 성취수준(A/B/C/D/E 등)을 체계적으로 정리하고 열람할 수 있는 웹 기반 플랫폼입니다. 교사들이 성취수준을 쉽게 확인하고 성적 처리 및 피드백에 활용할 수 있도록 돕는 것을 목적으로 합니다.

## 2. 디자인 및 기능 명세
- **비주얼 스타일:** 현대적이고 깨끗한 UI, 카드 기반 레이아웃, `oklch` 컬러 팔레트 활용.
- **주요 기능:**
  - 메인 화면: 중학교 교과별 선택 메뉴 (아이콘 및 카드 형태).
  - 교과 상세 화면: 선택한 교과의 성취기준 및 수준별 내용 표시.
  - 반응형 디자인: 데스크톱 및 모바일 최적화.
- **기술 스택:**
  - HTML5, Modern CSS (Container Queries, Cascade Layers).
  - Vanilla JavaScript (ES Modules, Web Components).
